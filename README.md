# React + Vite + Tailwind Project

## Steps Completed in This Project

1. **Create the React app**
   - Used the command:
     ```
     npm create vite@latest <project-name>
     ```
   - Replaced `<project-name>` with the chosen project name.

2. **Clean up the `src` folder**
   - Deleted all files inside the `src` folder except:
     - `index.css`
     - `app.jsx`
     - `main.jsx`
    

3. **Create the App component**
   - Added a new file called `App.jsx`.
   - Imported all necessary modules, including React and TailwindCSS.

4. **Install and configure TailwindCSS**
   - Followed the official TailwindCSS documentation on the "Get Started" page.
   - Installed required dependencies.
   - Configured `tailwind.config.js`.
   - Imported Tailwind directives into the project’s CSS file.

5. **Build the App layout**
   - Inside the `App` component, created a React fragment.
   - Added a `div` with the className `"App"`.
   - Used TailwindCSS to create:
     - A navbar
     - A heading
     - Three cards

6. **Run the project**
   - Started the development server with:
     ```
     npm run dev
     ```
   - Opened the project in the browser at `http://localhost:5173`.
